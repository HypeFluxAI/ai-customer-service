## 二、ReAct 循环 — 核心工作流

```
用户输入: "GPU 서버 가격이 어떻게 되나요?"
  │
  ▼
┌──────────────────────────────────────────────────────────┐
│                    ReAct 循环                             │
│                                                          │
│   ┌─────────────────────────────────────────────────┐    │
│   │ 1. THINK (推理)                                  │    │
│   │    "用户问 GPU 价格，我需要搜索知识库"             │    │
│   └─────────────────────┬───────────────────────────┘    │
│                         ▼                                │
│   ┌─────────────────────────────────────────────────┐    │
│   │ 2. ACT (行动)                                    │    │
│   │    调用工具: kb_search("GPU 서버 가격")           │    │
│   └─────────────────────┬───────────────────────────┘    │
│                         ▼                                │
│   ┌─────────────────────────────────────────────────┐    │
│   │ 3. OBSERVE (观察)                                │    │
│   │    工具返回: "RTX 4090 시간당 $0.3..."            │    │
│   └─────────────────────┬───────────────────────────┘    │
│                         ▼                                │
│   ┌─────────────────────────────────────────────────┐    │
│   │ 4. THINK (再推理)                                │    │
│   │    "找到信息了，可以回答用户"                      │    │
│   └─────────────────────┬───────────────────────────┘    │
│                         ▼                                │
│   ┌─────────────────────────────────────────────────┐    │
│   │ 5. RESPOND (回复)                                │    │
│   │    "RTX 4090 기준 시간당 $0.3입니다..."           │    │
│   └─────────────────────────────────────────────────┘    │
│                                                          │
│   * 循环可以多次 THINK → ACT → OBSERVE 直到完成任务     │
└──────────────────────────────────────────────────────────┘
```

## 三、工具调用详细流程

```
              GeminiClient                     Gemini API
                  │                                │
  用户消息 ──────▶│── sendMessageStream() ────────▶│
                  │                                │
                  │◀── 流式返回 ──────────────────│
                  │    [文本块 + 工具调用请求]       │
                  │                                │
                  ▼                                │
          CoreToolScheduler                        │
                  │                                │
                  │  状态流转:                       │
                  │  scheduled                      │
                  │     ▼                           │
                  │  validating (参数校验)            │
                  │     ▼                           │
                  │  ┌─ PolicyEngine 评估 ──┐       │
                  │  │ ALLOW → 直接执行      │       │
                  │  │ ASK_USER → 用户确认   │       │
                  │  │ DENY → 拒绝执行       │       │
                  │  └─────────────────────┘       │
                  │     ▼                           │
                  │  executing (执行工具)             │
                  │     ▼                           │
                  │  success / error                 │
                  │                                │
                  │── 工具结果 ─────────────────────▶│
                  │                                │
                  │◀── 继续流式回复 ────────────────│
                  │    [最终文本回复]                 │
                  ▼                                │
              输出给用户                             │
```

## 四、MCP Server 通信机制

```
┌──────────────┐         stdio (JSON-RPC 2.0)        ┌──────────────────┐
│              │◀═══════════════════════════════════▶│                  │
│  Gemini CLI  │         stdin / stdout               │  MCP Server      │
│              │                                      │  (Python 进程)   │
│  McpClient   │  ① initialize ──────────────────▶   │                  │
│  Manager     │  ◀── capabilities ──────────────    │  kb_server.py    │
│              │                                      │                  │
│              │  ② tools/list ──────────────────▶   │  ┌────────────┐  │
│              │  ◀── [kb_search, kb_teach, ...] ──  │  │ ChromaDB   │  │
│              │                                      │  │ 知识库      │  │
│              │  ③ tools/call ──────────────────▶   │  └────────────┘  │
│              │     {name: "kb_search",              │                  │
│              │      arguments: {query: "..."}}      │                  │
│              │  ◀── result ────────────────────    │                  │
│              │     {content: [{text: "..."}]}       │                  │
└──────────────┘                                      └──────────────────┘

通信协议:
  • JSON-RPC 2.0 over stdio
  • 请求: {"jsonrpc":"2.0", "id":1, "method":"tools/call", "params":{...}}
  • 响应: {"jsonrpc":"2.0", "id":1, "result":{...}}
```

## 六、一次完整训练对话的数据流

```
用户: "GPU 서버 가격이 어떻게 되나요?"
  │
  ▼
Gemini CLI → Gemini 2.5 Pro API
  │
  │  AI 推理: "需要查知识库"
  │
  ▼
Gemini CLI → MCP → kb_server.py → ChromaDB
  │                                   │
  │  kb_search("GPU 서버 가격")        │
  │                                   │
  │  ◀── 返回 5 条相关知识 ───────────┘
  │
  ▼
Gemini 2.5 Pro: 基于知识库生成回复
  │
  ▼
用户看到: "RTX 4090 기준 시간당 $0.3입니다. 자세한 사항은..."
  │
  │
用户: "틀렸어, 정확한 가격은 $0.5야"
  │
  ▼
Gemini CLI → MCP → kb_server.py
  │
  │  kb_correct(question="GPU 가격",
  │             wrong_answer="$0.3",
  │             correct_answer="$0.5")
  │                    │
  │                    ▼
  │              ChromaDB learned_qa 에 저장
  │              + data/training/ 로그 기록
  │
  ▼
用户看到: "✓ 수정 완료. RTX 4090 시간당 $0.5입니다."
  │
  │
  ▼  下次再问同样问题 → 知识库已更新 → 回答正确
```

## 七、安全权限模型

```
┌──────────────────────────────────┐
│         PolicyEngine             │
│                                  │
│  用户输入 → 工具调用请求           │
│         ↓                        │
│  ┌─────────────────────┐         │
│  │ 权限模式             │         │
│  │                     │         │
│  │ DEFAULT  → 写操作问  │         │
│  │ YOLO     → 全自动    │ ← Subagent 默认│
│  │ PLAN     → 只读      │         │
│  │ AUTO_EDIT→ 自动编辑  │         │
│  └─────────────────────┘         │
│         ↓                        │
│  ┌─────────────────────┐         │
│  │ 决策                 │         │
│  │ ALLOW    → 直接执行  │         │
│  │ ASK_USER → 弹窗确认  │         │
│  │ DENY     → 拒绝      │         │
│  └─────────────────────┘         │
│         ↓                        │
│  可选: 沙箱隔离                   │
│  • macOS Seatbelt                │
│  • Docker 容器                   │
│  • Podman 容器                   │
└──────────────────────────────────┘
```
